# Customer Churn Prediction

## Business Problem
Predict which customers are likely to leave a telecommunications company.

## Dataset
IBM Telco Customer Churn Dataset

## Tools
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Seaborn

## Model
Logistic Regression

## Results
Accuracy: 80.4%
Precision: 64.8%
Recall: 57.5%
ROC-AUC: 83.6%

## Business Recommendations
- Encourage long-term contracts
- Improve onboarding for new customers
- Target retention campaigns toward high-risk customers
